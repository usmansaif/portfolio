PakPoultry.pk is a fully automated poultry management system designed and developed from scratch using Laravel, PHP, Blade, jQuery, and Bootstrap. The platform offers a subscription-based service where users can receive daily SMS alerts tailored to their selected area and alert type, such as day-old chick rates, daily egg rates, daily chicken rates, and farm rates.

The system is built to collect data automatically by scraping and parsing information from multiple sources. It categorizes the data and sends personalized SMS alerts to subscribers based on their preferences. Additionally, it dynamically generates detailed price lists and posts them as rate updates on the Facebook page.

Key Features:
  • Automated SMS Alerts: Sends customized daily updates to subscribers based on selected areas and alert types.
  • Data Scraping & Categorization: Automatically scrapes data from various sources, parses it, and organizes it by type and region.
  • Dynamic Rate Lists: Creates dynamic price lists for different poultry categories, shared automatically on the platform and Facebook.
  • Admin Panel: Includes a robust admin interface for managing subscribers, data, and services. The admin can update SMS and email servers through an intuitive UI.
  • End-to-End Management: Developed and managed all front-end and back-end aspects of the application independently.
