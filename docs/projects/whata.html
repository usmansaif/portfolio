Project Overview

I developed a custom Android application for Whatapratha, a renowned food brand, to streamline its internal order delivery process. The app focuses on enhancing delivery accuracy and customer satisfaction by implementing a robust OTP-based verification system.

Key Features Delivered

  • Real-time delivery tracking and confirmation
  • Business-exclusive internal management interface
  • Offline-first architecture with SQLite for reliable data storage
  • REST API integration for centralized order management
