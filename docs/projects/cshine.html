A private Android mobile application designed for company use, enabling users to track pests through barcode scanning. The workflow begins with one user visiting a location to place barcodes for identification. When another user visits the same location, they use the app to scan the barcodes and record relevant data or submit values associated with each barcode.

Some Features:
  • Auto uninstall Trims application which is the installer of this app
