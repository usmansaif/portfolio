petfinder.pk

PetFinder.pk is a web application built to help reunite lost pets with their owners and support pet care. Users can report lost, found, or abandoned pets, with features like automatic poster generation, geolocation-based marking, and notification for reuniting pets. The platform also includes a directory of vets and pet stores with opening hours and services, plus a blog for pet care tips. It is fully optimized for SEO to ensure maximum reach.

Responsibilities:
  • Developed the application from scratch using Laravel, Blade, jQuery, and Bootstrap.
  • Implemented features like poster generation, area-based marking, and pet-owner notifications.
  • Created searchable directories for vets and pet stores to assist pet owners.
  • Integrated a blog for pet-related articles.
  • Optimized the platform for SEO and performance.
  • Managed development, testing, and deployment phases.
